# hooks
My personal configs and hooks

## 1、拉取代码

```sh
git clone https://github.com/MinsonLee/personal-config.git ~/personal-config
```

## 2、填写配置信息

1. 复制项目目录下的 conf.example 为 conf 文件

```sh
cp ./conf.example ./conf
```

2. 编辑 conf 文件，填写你的 Git 用户名/邮箱
3. 执行 `./install.sh`
